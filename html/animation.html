animation

Właściwość skrótowa animation łączy osiem oddzielnych właściwości animacji w jedną.

Przyjmuje ona po kolei od jednej do ośmiu wartości, kolejno: nazwę animacji, jej czas trwania, krzywą prędkości, opóźnienie, ilość wykonań, kierunek wykonywania, sposób dopełnienia i stan wstrzymania. Niektóre z nich mogą być pominięte, niektóre mogą też zostać wpisane w innej kolejności.

Style określone za pomocą właściwości skrótowych mogą być zmienione przez właściwości pojedyncze o wyższym priorytecie.

animation: nazwa czas_trwania krzywa_prędkości opóźnienie ilość_wykonań kierunek tryb_dopełnienia stan_wstrzymania;
// Zamiast pisać:
animation-name: testowa_animacja;
animation-duration: 5s;
animation-timing-function: linear;
animation-delay: 2.5s;
animation-iteration-count: infinite;
animation-direction: alternate;
animation-fill-mode: forwards;
animation-play-state: paused;

// Można użyć:
animation: test czas_trwania krzywa_prędkości opóźnienie ilość_wykonań kierunek tryb_dopełnienia stan_wstrzymania;
