transition

Chociaż do tworzenia skomplikowanych animacji wymagane są właściwości z rodziny animation, to aby stworzyć proste zmiany pojedynczych właściwości można użyć właściwości transition, do których należą -delay, -duration, -property i -timing-function.

Pozwalają one płynnie zmienić daną właściwość dla konkretnego elementu, jednak służą one tylko jako dodatek - same z siebie nie spowodują zmiany na stronie.

transition-duration: czas_przejścia;
transition-delay: długość_opóźnienia;
transition-property: zmieniana_właściwość;
transition-timing-function: krzywa_prędkości;
transition: właściwość czas_przejścia krzywa_prędkości opóźnienie;

Jeśli dla jednego elementu chcemy ustawić więcej niż jedną właściwość transition, należy to zrobić za pomocą właściwości skrótowej, wypisując kolejne zestawy wartości po przecinku. Zostało to zaprezentowane na poniższym przykładzie:

#box {
    height: 200px;
    transition: filter 5s linear, height 2s ease-in-out;
}

#box:hover {
    height: 320px;
    filter: hue-rotate(-110deg);
    transition-property: height;
    transition-duration: 2s;
    transition-timing-function: ease-in;
    transition-delay: 1s;
}
