animation-direction

Właściwość animation-direction określa kierunek, w jakim animacja jest wykonywana, czyli kolejność wykonywania instrukcji w niej zawartych.

Może ona poruszać się tylko do przodu, tylko do tyłu lub na zmianę, w przypadku ruchu na zmianę można wybrać kierunek rozpoczęcia. Wartością domyślną jest normal.

animation-direction: normal|reverse|alternate|alternate-reverse;
animation-direction: normal;

@keyframes kierunek {
    0% {
        transform: translate(0px,0px);
    }
    25% {
        transform: translate(300px,0px);
    }
    50% {
        transform: translate(300px,300px);
    }
    75% {

    }
    100% {
        transform: translate(0px,0px);
    }
