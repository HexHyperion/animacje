@keyframes

Reguła @kysframes jest używana do tworzenia animacji, stopniowo zmieniając jeden lub wiele styli CSS elementów, do których jest przypisana. Składnia tej reguły wygląda następująco:

@keyframes nazwa_animacji {moment_wykonania {właściwość: wartość;}}

W czasie trwania animacji zmiana może nastąpić dowolną liczbę razy i w dowolnym momencie. Moment zmiany stylu określamy za pomocą słów kluczowych from i to lub wartości podanych w procentach, np. 50%. Selektory początku i końca powinny być zawsze zdefiniowane.

Jeżeli do właściwości w środku tej reguły dodana zostanie reguła !important, to właściwość taka zostanie całkowicie zignorowana.

@keyframes test-1 {
    from {
        top: 50px;
        left: 50px;
    }
    to {
        top: 350px;
        left: 350px;
    }
}
