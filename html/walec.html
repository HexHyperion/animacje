Jak stworzyć walec?

Ta strona pokazuje, w jaki sposób można stworzyć element 3D przypominający walec za pomocą wyłącznie HTMLa i CSSa. Końcowy kształt obraca się wokół własnej osi, a na jego bokach umieszczony jest obraz. Przekształcając element .holder można dowolnie obracać walcem bez wpływu na zawartość.

<!-- 
Przydatne do stworzenia tej struktury mogą się okazać skróty Emmet: 

    div.face[style="--index:$"]*50

stworzy 50 div'ów o klasie face z wartościami index od 0 do 49 (przyda się później)
-->

<div class="holder">
    <div class="cylinder">
        <div class="face" style="--index:1"></div>
        <div class="face" style="--index:2"></div>
        <div class="face" style="--index:3"></div>
        ···
        <div class="face" style="--index:50"></div>
    </div>
</div>
:root {
  --pi: 3.14159265358979;
  --cylinder-width: 1000px;
  --face-count: 50;
  --face-deg: calc(360deg / var(--face-count));
  --face-width: calc(var(--cylinder-width) / var(--face-count));
  --face-shift: calc(var(--cylinder-width) / var(--pi) / 2);
}
.holder {
    transform-style: preserve-3d;
    transform: rotateX(-15deg);
}

.cylinder {
    height: 500px;
    transform-style: preserve-3d;
    position: relative;
}
.face {
    position: absolute;
    background-color: mediumaquamarine;
    height: 100%;
    width: 100px;        // tymczasowe, pokazuje .face'y
    opacity: 0.9;
    border-block: 2px solid #404080;
}

.face:nth-child(3n) {
    background-color: mediumseagreen;
}
.face:first-child {
    background-color: darkcyan;
}
.face {
    ···
    width: 100px;
    width: var(--face-width);
    transform: rotateY(calc(var(--face-deg) * var(--index)))
               translateZ(calc(var(--face-shift) - 4px));
}

.face:first-child {
    border-left: 2px solid #404080;
}
.cylinder {
    ···
    animation-name: spin;
    animation-duration: 20s;
    animation-timing-function: linear;
    animation-iteration-count: infinite;
}
    
@keyframes spin {
    0% {
        transform: rotateY(0deg);
    }
    100% {
        transform: rotateY(360deg);
    }
}
.face {
    ···
    background: url("../img/galaxy.png");
    background-size: 1000px;
    background-position: calc(var(--index) *
                         var(--face-width) * -1)
}

<!-- Skończone! Od teraz walec powinien mieć obrazek i obracać się wokół własnej osi -->
