animation-delay

Właściwość animation-delay opóźnia czas rozpoczęcia wykonania animacji, może też zmieniać stopień wykonania, w jakim zostanie ona uruchomiona.

Przyjmuje ona czas w sekundach lub milisekundach, dopuszczalne są zarówno wartości dodatnie jak i ujemne, domyślną jest 0s. W przypadku gdy wartość jest ujemna, animacja rozpocznie się tak, jakby była odtwarzana już przez podany czas.

animation-delay: długość_opóźnienia;
animation-delay: 3s;

@keyframes opoznienie {
    0% {
        top: 0px;
        left: 0px;
    }
    100% {
        top: 380px;
        left: 380px;
    }
}
