animation-name, animation-duration

Właściwość animation-name określa nazwę animacji zdefiniowaną w regule @keyframes, niejako przypisując animację do elementu.

Właściwość animation-duration określa ile czasu powinna zająć animacja, aby ukończyć jeden cykl (0-100%). Jeśli czas ten nie został określony, animacja nie wykona się, ponieważ domyślną wartością tej właściwości jest 0s.

animation-name: nazwa_animacji;
animation-duration: czas_trwania;

Pokazana poniżej animacja normalnie wykonałaby się tylko jeden raz, ponieważ nie jest określona ilość wykonań, jednak dla lepszego wyglądu została ona zapętlona.

animation-name: obroc;
animation-duration: 5s;
    
@keyframes obroc {
    from {transform: rotate(0deg);}
    to {transform: rotate(360deg);}
}

@keyframes skaluj {
    0% {transform: scale(1);}
    50% {transform: scale(1.5);}
    100% {transform: scale(1);}
}
