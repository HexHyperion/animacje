animation-composition

Właściwość animation-composition jest używana do określenia zachowania elementu w sytuacji, w której kilka różnych animacji modyfikuje go jednocześnie.

Domyślną wartością jest replace, powoduje ona domyślne dla języka CSS nadpisanie mniej ważnej animacji. Możliwe jest jednak ustawienie, aby zostały one zastosowane niezależnie, lub żeby późniejsza animacja była zastosowana do efektu poprzedzającej.

animation-composition: replace|add|accumulate;

Poniższe pudełko jest na stałe obrócone o 30 stopni, a po najechaniu przesuwa się o 200px w dół:

animation-composition: replace;

#box{
    transform: rotate(30deg);
}
#box:hover{
    animation: laczenie 1s ease-in-out forwards;
}

@keyframes laczenie {
    0% {filter: translateY(0px);}
    100% {filter: translateY(200px);}
}
